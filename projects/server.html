ASCII Unicode

Early Character encoding, represents 128 characters with 7 bits (US Only).

Unicode

Universal character encoding, which supports many more characters across many languages and includes ASCII.

UTF-8

unicode that is variable in length, making it more efficient and more compact.

URL

An address on the web which leads to webpage.

Domain Name

Key element in web address, maps IP address using DNS.

HTTP

Data transfer protocol on web.

HTTPS

secure data transfer (encrypted).

IP Address

unique identifier assigned all devices, used to communicate across the Network.

DNS

translates domain into an ip address.

W3C Recommendations
